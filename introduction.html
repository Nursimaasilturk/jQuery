What is The jQuery?

Jquery is a JS library that most common used in the world. With jquery, you can do so many awesome things like animations, menu, banners, logos etc. Moreover,these are just the beginning. Below we have listed the things done with jquery.

  • HTML/DOM Manipulation
  • CSS Manipulation
  • AJAX
  • Event Handling
  • Animations,Effects

In addition to these, jquery is a great blessing for us in terms of its small size, open source, cross-browser support and easy to learn. Let's not talk too much and let's start coding!!!

Adding jQuery to your webpage

You can apply jquery to your webpage by downloading jquery or including from a CDN .

There are two version of jquery. You can download both version from jQuery.com. The jQuery library is a single JavaScript file, you can reference it with HTML <script> tags inside the <head> section.

<head>

<script src="jquery-3.6.0.min.js"><script>"

<head>

If you don't want to download jquery, you can include it from CDN.

<head>

<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.0/jquery.min.js"><script>"

<head>

jQuery Syntax

$(selector).action()

A $ sign to define jQuery

A (selector) to "find" HTML elements

A action()to be performedon elements

$(document).ready(function(){
// jQuery methods go here...
});

The Document Ready Event

This is to prevent any jQuery code from running before the document is finished loading (is ready).

$(function(){
// jQuery methods go here...
});

This is shorter method for document ready event.

You should use this syntax due to be easier to understand than other.
